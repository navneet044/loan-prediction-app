# loan-prediction-app
A machine learning–based web application that predicts whether a user is eligible for a loan based on financial and personal inputs. The system uses a trained classification model integrated with a Flask backend to process user inputs, apply preprocessing techniques, and generate real-time predictions through an interactive web interface.
